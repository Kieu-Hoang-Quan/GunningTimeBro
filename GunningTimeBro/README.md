# GunningTimeBro

Simple Java game project using Java + IntelliJ.

## Quick start

1. Build using your IDE (IntelliJ) or javac for simple compilation.
2. Run `Main` class located at `src/main/Main.java`.

## Notes

- This repository intentionally ignores IDE files and build artifacts. See `.gitignore` for more info.
- If you want to run locally from command line, compile the sources and run `Main`.
